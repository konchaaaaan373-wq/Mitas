Necoのサービスに関するよくある質問をまとめました

サービス全般

Necoはどのようなサービスですか？

+
NecoはAI技術を活用した医療人材紹介サービスです。医師・看護師の専門性を定量的に評価し、従来の求人マッチング方式ではなく、AIコンシェルジュが一人ひとりに最適な求人をご提案します。特に在宅医療分野に強みを持ち、訪問看護が初めての方も安心してご利用いただけるサポート体制を整えています。

利用料金はかかりますか？

+
求職者（医師・看護師）の方は完全無料でご利用いただけます。登録料、相談料、紹介手数料など、一切の費用はかかりません。求人企業・医療機関のみ、採用が決定した際に紹介手数料をお支払いいただく仕組みとなっています。

他の人材紹介サービスとの違いは何ですか？

+
Necoは以下の3点で従来のサービスと大きく異なります。①専門性の定量評価：経験年数だけでなく、症例数や手技の習熟度など具体的なスキルを評価します。②AIコンシェルジュ：単なる求人情報の提供ではなく、AIが一人ひとりに最適な案件をインテリジェントに提案します。③長期的サポート：就職前の研修から手技獲得、入職後のフォローまで一貫してサポートします。

登録・利用方法

登録に必要なものは何ですか？

+
初回登録には、メールアドレスのみで仮登録が可能です。その後、専任コンシェルジュとの面談時に、医師免許証または看護師免許証、職務経歴書をご準備ください。訪問看護の場合は、対応可能な処置内容などをヒアリングさせていただきます。

登録後の流れを教えてください

+
①メールアドレス登録 → ②専任コンシェルジュとの面談（オンライン可） → ③専門性の評価・希望条件のヒアリング → ④AIによる最適な求人マッチング → ⑤求人のご紹介 → ⑥面接調整・サポート → ⑦採用決定・入職サポート という流れになります。平均25〜30日程度でマッチングが成立しています。

地方在住でも利用できますか？

+
はい、全国どこからでもご利用いただけます。面談はオンラインで実施可能ですので、お住まいの地域に関わらずサービスをご利用いただけます。また、UIターン希望の方もサポートしておりますので、お気軽にご相談ください。

医師向けサービス

どのような求人がありますか？

+
若手〜中堅医師向けの非常勤（バイト）案件を中心に取り扱っています。外来診療、当直、訪問診療、健診など、多様な勤務形態の求人がございます。専門科別では、内科系、外科系、救急科など幅広くカバーしており、ご希望に応じて最適な案件をご紹介します。

専門性の評価とはどのようなものですか？

+
経験年数だけでなく、症例数、手技の習熟度、専門医資格の有無、学会発表・論文の実績などを総合的に評価します。これにより、あなたの市場価値を正確に把握し、適正な条件での求人紹介が可能になります。

看護師向けサービス

訪問看護が初めてでも大丈夫ですか？

+
はい、訪問看護が初めての方も安心してご利用いただけます。就職前の研修サポート、同行訪問支援、必要な手技の獲得サポートなど、入職前から入職後まで一貫してサポートします。在宅医療未経験の方の採用実績も多数ございます。

処置スキルの評価とは何ですか？

+
エンゼルケア、ストマ交換、褥瘡ケア、吸引、点滴管理、ポート管理、経管栄養・胃瘻管理、バルーン交換、在宅人工呼吸器対応など、訪問看護で必要とされる処置について、対応可能か、経験症例数はどの程度かなどを詳細に評価します。これにより、あなたのスキルに最適な求人をマッチングします。

夜間オンコールが不安ですが、サポートはありますか？

+
はい、オンコール対応に不安がある方向けに、経験豊富な看護師による同行支援や、24時間相談可能な体制を整えている訪問看護ステーションを優先的にご紹介します。また、段階的にオンコール対応に慣れていけるようなキャリアプランもご提案可能です。

個人情報・セキュリティ

個人情報は安全に管理されていますか？

+
当社は個人情報保護法を遵守し、厳格な安全管理措置を講じています。個人情報は暗号化して保管し、アクセス制限を設けるなど、漏洩防止策を徹底しています。また、求人企業への情報提供は、必ず事前にご本人の同意を得た上で行います。詳しくはプライバシーポリシーをご確認ください。

現在の勤務先にバレませんか？

+
ご安心ください。ご本人の同意なく、現在の勤務先や第三者に情報を開示することは一切ございません。求人企業への情報提供も、必ず事前に確認させていただきます。秘密厳守でサポートいたします。

他にご不明な点がございましたら

お気軽にお問い合わせください。専任コンシェルジュが丁寧にご対応いたします。
